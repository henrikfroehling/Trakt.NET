Modules

The modules in the library represent each a section in the official Trakt API documentation. You can not create a new instance of any module. The only way to access a module is through a TraktClient instance.

Authentication

This module contains all methods, that are described in Trakt API Authentication - OAuth and in Trakt API Authentication - Devices sections.

Access this module via TraktClient.Authentication.
