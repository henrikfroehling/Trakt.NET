    Class TraktAuthenticationModule

    Provides access to OAuth and device authentication and authorization.

    This module contains all methods of the "Trakt API Doc - Authentication - OAuth" section and the "Trakt API Doc - Authentication - Devices" section.

    Device

    IsAuthorized
