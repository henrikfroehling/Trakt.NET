    Class TraktSeasonsModule

    Provides access to data retrieving methods specific to seasons.

    This module contains all methods of the "Trakt API Doc - Seasons" section.

    Name Description
    GetAllSeasonsAsync(String, TraktExtendedInfo, String, CancellationToken)

    Gets the ITraktSeasons in a show with the given Trakt-Show-Id or -Slug.

    OAuth authorization not required.

    See "Trakt API Doc - Seasons: Summary" for more information.

    GetMultipleSeasonsAsync(TraktMultipleSeasonsQueryParams, CancellationToken)

    Gets multiple different seasons at once in a show with the given Trakt-Show-Id or -Slug.

    OAuth authorization not required.

    See "Trakt API Doc - Seasons: Season" for more information.

    See also GetSeasonAsync(String, UInt32, TraktExtendedInfo, String, CancellationToken).
    GetSeasonAsync(String, UInt32, TraktExtendedInfo, String, CancellationToken)

    OAuth authorization not required.

    See "Trakt API Doc - Seasons: Season" for more information.
